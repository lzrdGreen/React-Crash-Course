Lesson 12: Styling in React

Objective

Explore and apply different techniques for styling React components.

Topics Covered

  • Inline styles: Apply styles directly to JSX elements using JavaScript objects. This approach is simple for small, dynamic styles but can become less maintainable for larger projects.
  • CSS Modules: Write CSS in separate files with locally scoped class names, preventing naming collisions and improving CSS organization. Import these styles into your components.
  • CSS-in-JS Solutions: Use JavaScript to style your components, allowing you to write CSS within your JavaScript code and benefit from JavaScript's dynamic capabilities. Libraries like styled-components offer features like theming and component composition.

Hands-On

Style a component using CSS Modules and styled-components.

Example Code

  
    // Using CSS Modules
    import styles from "./Button.module.css";

    function Button() {
      return <button className={styles.button}>Click Me</button>;
    }

    export default Button;

    /* Button.module.css */
    .button {
      background-color: blue;
      color: white;
      padding: 10px 20px;
      border: none;
      border-radius: 5px;
    }

    // Using styled-components
    import styled from "styled-components";

    const StyledButton = styled.button<
      background-color: blue;
      color: white;
      padding: 10px 20px;
      border: none;
      border-radius: 5px;
    >;

    function App() {
      return <StyledButton>Click Me</StyledButton>;
    }

    export default App;
