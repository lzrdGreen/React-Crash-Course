Lesson 13: Debugging and Testing in React

Objective

Learn how to debug React applications effectively and write tests to ensure reliability.

Topics Covered

  • React Developer Tools: Inspecting components and state.
  • Debugging techniques (e.g., using console.log, breakpoints).
  • Introduction to testing with Jest and React Testing Library.
  • Snapshot testing and unit testing React components.

Hands-On

  1. Use React Developer Tools to inspect a component's props and state.
  2. Write a simple unit test for a functional component using Jest and React Testing Library.
  3. Test rendering and interactions of a button component.

Example Code

Debugging with React Developer Tools

Install React DevTools as a browser extension and inspect the props and state of components:

        // Inspect components in the React tab of your browser DevTools
        console.log("Debugging Component:");

Testing with Jest and React Testing Library

Install Jest and React Testing Library:

        npm install --save-dev jest @testing-library/react @testing-library/jest-dom

Write a test for a simple button component:

          import { render, fireEvent } from "@testing-library/react";
          import "@testing-library/jest-dom";
          import Button from "./Button";
        
          test("Button renders and responds to clicks", () => {
            const handleClick = jest.fn();            
            const { getByText } = render(<Button onClick={handleClick}>Click Me</Button>);
            
            const button = getByText("Click Me");
            fireEvent.click(button);
            expect(handleClick).toHaveBeenCalledTimes(1);
          });
