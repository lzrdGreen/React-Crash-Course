Lesson 3: JavaScript Objects & Arrays

Why Learn Objects & Arrays?

React props and state often involve working with objects and arrays.

What You'll Learn

  • Object destructuring: This allows you to extract values from objects into distinct variables. For example, const { name, age } = person; extracts the name and age properties from the person object.

  • Array destructuring: This allows you to extract values from arrays into distinct variables. For example, const [first, second] = myArray; extracts the first and second elements from the myArray array.

  • Array methods (map, filter, reduce, find): These provide powerful ways to transform and manipulate arrays. For example, const doubled = numbers.map(num => num * 2); creates a new array with each number doubled.

  • Spread/rest operators: The spread operator (...) expands iterables (like arrays and objects) into individual elements, while the rest parameter collects remaining arguments into an array. For example, const newArray = [...oldArray, newItem]; creates a new array by copying oldArray and adding newItem.
