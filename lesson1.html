Arrow Functions

A shorter syntax for writing functions.

const add = (a, b) => a + b; // Returns the sum of a and b

Key Features: Lexical this binding: Arrow functions inherit the this value from the surrounding (enclosing) scope. Shorter code, and implicit return for single-line functions.

this in Action

Here's an example demonstrating how this behaves differently in regular and arrow functions:


          const myObject = {
            value: 42,
            regularFunction: function() {
              console.log("Regular function this:", this.value);
            },
            arrowFunction: () => { // Note the arrow function here as well
              console.log("Arrow function this:", this.value);
            },
          };
          
          myObject.regularFunction(); // Output: 42
          myObject.arrowFunction(); // Output: undefined (in strict mode), or something else like window in non-strict mode. This is the key difference!
          
          const anotherObject = {value:100}
          anotherObject.regularFunction = myObject.regularFunction;
          anotherObject.regularFunction(); // Output: 100
          anotherObject.arrowFunction = myObject.arrowFunction;
          anotherObject.arrowFunction(); // Output: undefined (in strict mode), or something else like window in non-strict mode. This is the key difference!

Default Parameters

Set default values for function parameters when no arguments are provided.

Example: 

        function greet(name = "Guest") {
          return `Hello, ${name}`;
        }

Template Literals

Use backticks (``) for strings that allow embedding variables and expressions.

Example:

        const name = "John";
        console.log(`Hello, ${name}!`);
