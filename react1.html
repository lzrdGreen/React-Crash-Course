React Basics

Lesson 1: Introduction

Objective

Understand the core building blocks of React, including how to organise code using Modules and Import/Export.

Topics Covered

  • What is React? A JavaScript library for building user interfaces (UIs). React allows you to create reusable UI components, making it easier to build and maintain complex web applications.
  • JSX: A syntax extension to JavaScript that lets you write HTML-like code within your JavaScript. JSX makes it easier to visualise and structure your UI.
  • Components: The fundamental building blocks of React UIs. Components are reusable pieces of code that render HTML.Functional vs. Class components: We will primarily focus on functional components.
  • Props (Properties): A way to pass data from parent components to child components. Props are read-only from the child component's perspective.
  • Modules and Import/Export: A way to organise your React code into separate files (modules) and share code between them. import statements bring code from other modules into your current file, while export statements make code available to be used elsewhere. This keeps your codebase organized and maintainable.

Hands-On

  1. Create a basic App component.
  2. Add a child component, pass data via props, and display it.
  3. Split components into separate files and use import and export to organize the code.

Example Code A

Inline Components:

        function Welcome(props) {
        return <h1>Hello, {props.name}!</h1>;
        }

        function App() {
        return <Welcome name="React Learner" />;
        }

        export default App;

Example Code B

Separate Components:

Step 1: Create a Welcome.js file:

export function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

Step 2: Use import in the App.js file:

import { Welcome } from './Welcome';

function App() {
  return <Welcome name="React Learner" />;
}

export default App;
