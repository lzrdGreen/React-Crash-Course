React Basics

Lesson 1: React Basics

Objective

Understand the core building blocks of React.

Topics Covered

  • What is React? Component-based UI library.
  • JSX: Writing HTML-like syntax in JavaScript.
  • Components: Functional vs. Class components (focus on Functional).
  • Props: Passing data between components.

Hands-On

  1. Create a basic App component.
  2. Add a child component, pass data via props, and display it.

Example Code

        function Welcome(props) {
        return <h1>Hello, {props.name}!</h1>;
        }

        function App() {
        return <Welcome name="React Learner" />;
        }

        export default App;
