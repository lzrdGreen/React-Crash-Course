React Basics

Lesson 2: State and Event Handling

Objective

Manage component state and handle user interactions.

Topics Covered

  • useState Hook: Managing state in functional components.
  • Event Handling: Handling user input, button clicks, etc.
  • Two-Way Binding: Using state to track and display user input.

Hands-On

  1. Create a simple counter app.
  2. Add an input field that updates dynamically as the user types.

Example Code

import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

function App() {
  return <Counter />;
}

export default App;
