Lesson 2: State and Event Handling

Objective

Manage component state and handle user interactions.

Topics Covered

  • useState Hook: Managing state in functional components. This allows you to add state to your functional components, triggering re-renders when the state changes.

  • Event Handling: Handling user input, button clicks, and other browser events. This involves attaching event listeners to JSX elements.

  • Handling Form Inputs (Controlled Components): In React, form elements are typically controlled by React's state. This means that the input's value is driven by the state, and changes to the input update the state. This enables validation, dynamic updates, and more. For example: 

    
    import { useState } from 'react';
    
    function MyInput() {
      const [inputValue, setInputValue] = useState('');
    
      const handleChange = (event) => {
        setInputValue(event.target.value);
      };
    
      return (
        <input type="text" value={inputValue} onChange={handleChange} />
      );
    }

Hands-On

  1. Create a simple counter app.

  2. Add an input field that updates dynamically as the user types.

Example Code

import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

function App() {
  return <Counter />;
}

export default App;
