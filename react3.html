Lesson 3: Lists and Conditional Rendering

Objective

Work with dynamic data and render it conditionally.

Topics Covered

  • Rendering Lists: Using .map() to render arrays as components.
  • Keys in React: Importance of unique keys for list items.
  • Conditional Rendering: Using if or ternary operators to show/hide elements.

Hands-On

  1. Display a list of items dynamically.
  2. Show a message based on a condition (e.g., "No items found").

Example Code

function ItemList({ items }) {
  return (
    <div>
      {items.length === 0 ? (
        <p>No items found!</p>
      ) : (
        <ul>
          {items.map((item, index) => (
            <li key={index}>{item}</li>
          ))}
        </ul>
      )}
    </div>
  );
}

function App() {
  const items = ["Learn React", "Practice JSX", "Build Components"];
  return <ItemList items={items} />;
}

export default App;
