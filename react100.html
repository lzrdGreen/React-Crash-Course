React Basics

Lesson 1: Introduction

Objective

Understand the core building blocks of React, including Modules and Import/Export.

Topics Covered

  • What is React? Component-based UI library.
  • JSX: Writing HTML-like syntax in JavaScript.
  • Components: Functional vs. Class components (focus on Functional).
  • Props: Passing data between components.
  • Modules and Import/Export: Organizing React code.

Hands-On

  1. Create a basic App component.
  2. Add a child component, pass data via props, and display it.
  3. Split components into separate files and use import and export to organize the code.

Example Code

Step 1: Create a Welcome.js file:

export function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}

Step 2: Use import in the App.js file:

import { Welcome } from './Welcome';

function App() {
  return <Welcome name="React Learner" />;
}

export default App;
