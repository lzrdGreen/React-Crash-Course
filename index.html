Assumed Knowledge (JS)

There are numerous great courses on JavaScript (as well as HTML and CSS). The first part of this tutorial doesn't provide an overview of JS, instead it acts as a refresher of the crucial JS concepts for React. Here is a good course on JS by W3Schools: JavaScript Tutorial. They also have React Tutorial.

In any case, I assume you already know this:

  • Variables (let, const vs. var).

  • Basic conditionals (if/else).

  • Basic loops (for, while).

Setup Instructions

  • Install Node.js, npm, then you can create a React app (npx create-react-app).

  • As a quick fix:

  • Run your JS code in your browser's console. Alternatively, try Online JavaScript Playgrounds (Like CodePen, JSFiddle, StackBlitz, etc)
