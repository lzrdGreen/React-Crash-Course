Lesson 4: Basic Routing

Objective

Navigate between pages in a React app.

Topics Covered

  • React Router: Install and use react-router-dom.
  • Setting Up Routes: Create links and switch between pages.

Hands-On

  1. Add a Home page and an About page.
  2. Use BrowserRouter, Routes, and Link.

Example Code

import { BrowserRouter as Router, Routes, Route, Link } from "react-router-dom";

function Home() {
  return <h1>Home Page</h1>;
}

function About() {
  return <h1>About Page</h1>;
}

function App() {
  return (
    <Router>
      <nav>
        <Link to="/">Home</Link>
        <Link to="/about">About</Link>
      </nav>
      <Routes>
        <Route path="/" element={} />
        <Route path="/about" element={} />
      </Routes>
    </Router>
  );
}

export default App;
