JavaScript Essentials for React

Lesson 1: Functions & ES6+ Syntax

Why Learn Modern JavaScript?

React heavily uses modern JavaScript, especially ES6+ syntax, for functional components and hooks.

What You'll Learn

  • Arrow Functions
  • Default Parameters
  • Template Literals

Arrow Functions

A shorter syntax for writing functions.

Example: 

const add = (a, b) => a + b; // Returns the sum of a and b

Key Features: No binding of this, shorter code, and implicit return for single-line functions.

Default Parameters

Set default values for function parameters when no arguments are provided.

Example: 

        function greet(name = "Guest") {
          return `Hello, ${name}`;
        }

Template Literals

Use backticks (``) for strings that allow embedding variables and expressions.

Example:

        const name = "John";
        console.log(`Hello, ${name}!`);

Interactive Example

Original Function:

      function multiply(x, y) {
        return x * y;
      }

Task: Rewrite the above function as an arrow function and test it in your browser's console!
