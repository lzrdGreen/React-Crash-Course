Lesson 7: Lifecycle Methods

Objective

Understand React's component lifecycle and how to manage side effects using useEffect in functional components.

Topics Covered

  • The component lifecycle and its relevance.
  • useEffect: Handling side effects in functional components.
  • Cleanup functions within useEffect.

Hands-On

Create a component that fetches data on mount and cleans up resources on unmount.

Example Code

    
      import { useState, useEffect } from "react";

      function DataFetcher() {
        const [data, setData] = useState(null);

        useEffect(() => {
          const fetchData = async () => {
            const response = await fetch("https://jsonplaceholder.typicode.com/posts");
            const result = await response.json();
            setData(result);
          };

          fetchData();

          return () => {
            console.log("Cleanup on unmount");
          };
        }, []);

        return (

{JSON.stringify(data, null, 2)}

          

        );
      }

      export default DataFetcher;
